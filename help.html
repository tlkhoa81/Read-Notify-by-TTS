The app 'Read Notify by TTS' allows to config list app to read notification by TTS

Features:
  - Allow to manage list of apps to thread notifications to read by text to speech (view list, add, and modify, delete) 
  - Allow to manage list/ of notifications log (view list, detail, delete, delete all) 
  - Allow to modify setting to config language to read by text to speech 
  - Allow to share app
  - Allow to view help       

Example: I want to read the increase or decrease amount if any of SHB bank
  - Go to function Manage apps  
  - Add app 'SHB SAHA'
  - Transfer 5000 VND to SHB account to hear the balance increase
